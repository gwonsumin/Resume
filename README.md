# Portfolio Website

## Overview
이 프로젝트는 UX/UI 디자이너 포트폴리오 웹사이트입니다.

사용자의 상태를 이해하고,
그 흐름을 선택 가능한 경험으로 구조화하는 것을 목표로 설계했습니다.

## Concept
- State Interface
- 사용자 상태 기반 UX 설계
- 선택 중심 인터페이스 구조

## Tech Stack
- React
- SCSS
- TypeScript, Vite, React Router

## Structure
```bash
/src
  /components
  /pages
  /assets
    /logo
    /cursor
    /images
```

## Features
- Hero 중심 타이포 레이아웃
- 커스텀 커서 인터랙션
- 카드 기반 프로젝트 구조
- 섹션 간 스크롤 UX

## Design System
- Color
  - Main: #FF785D
  - Sub: #0AA5A5
  - Background: #FFF4EB
  - Line: #3B2F2F

- Typography
  - Inter (영문)
  - Pretendard (한글)

## Getting Started
```bash
npm install
npm run dev
```

## Notes
이 프로젝트는 단순 UI 구현이 아닌,
UX 흐름과 인터페이스 구조를 중심으로 설계되었습니다.
